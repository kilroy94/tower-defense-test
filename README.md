# RTS Prototype

Godot 4 prototype for RTS-style camera controls, worker commands, building placement, and command-card UI.

## Project Structure

- `assets/`: art, audio, fonts, and temporary source assets.
- `scenes/`: Godot scenes grouped by domain.
- `scripts/`: GDScript code grouped by domain.
- `data/`: focused JSON definitions for units, buildings, waves, and tuning.
- `docs/`: project memory, roadmap, mechanics, decisions, tasks, and balance notes.
- `tests/`: future automated and manual test support.
- `builds/`: local build output.

## Entry Point

- `res://scenes/core/main.tscn`
